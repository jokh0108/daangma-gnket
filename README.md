# 당마근켓

## 개요

당마근켓은 스스로 만들어보는 웹 기반 당근마켓 프로젝트입니다.

당근마켓을 따라 만들어보면서 설계부터 배포까지의 과정을 전부 경험해보고자 합니다. 인턴 때 했던 경험이나 학습했던 부분들을 최대한 적용해보고 다양한 스택들을 사용하면서 그것들이 왜 필요한지 직접 느껴보고 어떻게 사용하는지 익숙해질 예정입니다.
